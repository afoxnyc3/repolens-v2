# Repolens v2 - Product Specification

## Overview
Repolens v2 is an AI-native repository intelligence platform that transforms codebases into structured, token-efficient context for LLMs and humans.

## Core Capabilities
- Repo ingestion & filtering
- File classification & ranking
- Summarization (repo, directory, file)
- Context packaging (task-based)
- AI task execution with human review

## Architecture
Repo → Ingestion → Classification → Summarization → Context Packaging → AI Task → Output

## MVP Scope
- Read-only repo intelligence
- Context generation
- Task templates (analyze, summarize, refactor prep)
- Model abstraction (Anthropic/OpenAI)

## Future
- RAG
- Patch generation
- Agent workflows
